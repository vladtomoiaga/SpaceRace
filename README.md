## About the Project

This small data analysis project focuses on space mission launch data stored in a CSV file (`mission_launches.csv`).  
It demonstrates:
- How to perform **data cleaning** and **exploration** using **pandas**
- How to visualize data with **Matplotlib**, **Seaborn**, and **Plotly**
- How to extract insights such as the number of launches per organization or the proportion of active vs. retired rockets.

### Built With

The following technologies and libraries are used:

* Python
* Pandas
* Matplotlib
* Seaborn
* Plotly
* ISO3166 (for country code conversions)

Follow the steps below to set up the environment and run the project:
